# Cocozap Server

Built with Node.js, Express and MongoDB

### Setup

Create an .env file
`touch .env`

Insert on the `.env` file the PORT and MongoDB url collection access

```
Ex:
  PORT=3000
  MONGO_URL=mongodb+srv://example
  URL=http://localhost
```

Run Yarn scripts

install dependencies

```
yarn install
```

run server

```
yarn start
```
